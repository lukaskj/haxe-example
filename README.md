# Install all dependencies

```bash
$ haxelib newrepo
$ haxelib install all --always
```

# Run
```bash
$ haxe run.hxml
```

# Build

## Executable (from Neko)
```bash
$ haxe build-executable.hxml
```

## C++
```bash
$ haxe build-cpp.hxml
```

## Java
```bash
$ haxe build-java.hxml
```

## JavaScript
```bash
$ haxe build-js.hxml
```

## Python
```bash
$ haxe build-python.hxml
```

## PHP
```bash
$ haxe build-php.hxml
```

## Neko runtime
```bash
$ haxe build-neko.hxml
```
